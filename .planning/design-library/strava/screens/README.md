# Strava — reference screens

Drop Strava screenshots here (or into the chat and I'll file them). Priority
screens for the activity/health work, most useful first:

1. **Activity detail** — the stat block (distance/time/pace/elevation) hierarchy.
2. **Post-activity summary** — the completion/celebration moment.
3. **Weekly / training progress** — how activity trends are visualized.
4. *(optional)* **Feed card + kudos** — for the later social layer.

Suggested naming: `activity-detail.png`, `post-activity.png`,
`weekly-progress.png`, `feed-kudos.png`.

Once screens are here, `../patterns.md` gets updated from the real pixels
(spacing, type scale, color, chart treatment) — replacing the current analysis
that was reconstructed from public descriptions.
